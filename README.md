## Tech Stack

- **Frontend**: React 19, Vite 6, Tailwind CSS 3
- **Routing**: React Router 6
- **State Management**: Zustand
- **Icons**: Lucide React
- **Charts**: Recharts
- **Themes**: next-themes
- **TypeScript**: Full type safety
