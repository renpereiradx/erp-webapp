# Design System: ERP WEBAPP
**Project ID:** 693252806641531510

## 1. Visual Theme & Atmosphere
El sistema de diseño para el ERP WEBAPP se inspira en una estética moderna, limpia y altamente funcional. La atmósfera es de precisión profesional, utilizando un sistema de rejilla modular y bordes suaves que transmiten confianza y claridad.

## 2. Color Palette & Roles
* **Primary Blue (#137fec):** Utilizado para botones de acción principal, estados activos de navegación y destacados de marca.
* **Surface Background (#FFFFFF):** El color de fondo principal para tarjetas y contenedores en modo claro.
* **Text Main (#1F2937):** Color de texto principal para legibilidad óptima.
* **Text Secondary (#6B7280):** Color para etiquetas, descripciones y metadatos de menor jerarquía.
* **Success Green (#10B981):** Para indicadores de estado positivo (pagado, al día).
* **Error Red (#EF4444):** Para alertas críticas y saldos vencidos.

## 3. Typography Rules
* **Font Family:** Inter (Sans-serif).
* **Weights:** 
    * Bold para encabezados de sección y métricas clave.
    * Regular para cuerpo de texto y etiquetas.
    * Semi-bold para navegación y estados de botones.

## 4. Component Stylings
* **Buttons:** Bordes redondeados de 8px (ROUND_EIGHT), con un sombreado suave para dar profundidad. Las acciones primarias usan el azul `#137fec`.
* **Cards/Containers:** Esquinas con radio de 8px. Fondo blanco puro con una sombra difusa muy tenue para separar los elementos del fondo de la página.
* **Tables:** Cabeceras con fondo sutil y texto en mayúsculas pequeñas. Filas con separación clara y estados de hover delicados.

## 5. Layout Principles
* **Spacing:** Basado en una escala de 8px. Márgenes generosos de 32px para secciones principales.
* **Alignment:** Estructura centrada en una rejilla de 12 columnas para dashboards complejos.
